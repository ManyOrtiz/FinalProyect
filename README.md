This project uses the RWF-2000 dataset, a large-scale collection of surveillance videos labeled as Fight and NonFight.
The dataset is publicly available on Kaggle:

RWF-2000 Dataset (Kaggle)
https://www.kaggle.com/datasets/vulamnguyen/rwf2000

The dataset contains real-world surveillance videos and is not included in this repository due to its large size (~12 GB).
